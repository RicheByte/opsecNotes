### **Reconnaissance & OSINT (Open-Source Intelligence)**

1.  **theHarvester**: Used to gather emails, subdomains, hosts, employee names, and banners from public sources.
2.  **Maltego**: A powerful OSINT and link analysis tool for graphing relationships between data points.
3.  **Recon-ng**: A full-featured web reconnaissance framework written in Python.
4.  **Shodan**: A search engine for Internet-connected devices. (Uses API)
5.  **Amass**: In-depth DNS enumeration and network mapping tool.
6.  **SpiderFoot**: Automates the process of gathering OSINT from over 100 data sources.
7.  **FOCA (Fingerprinting Organizations with Collected Archives)**: Discovers metadata and hidden information in documents.
8.  **Sherlock**: Hunts down social media accounts by username across various platforms.

### **Vulnerability Scanners**

9.  **Nessus**: One of the most comprehensive and popular commercial vulnerability scanners.
10. **OpenVAS (Greenbone Vulnerability Manager)**: The powerful, open-source sibling of Nessus.
11. **Nexpose (Rapid7 InsightVM)**: A commercial vulnerability management solution that integrates with Metasploit.
12. **Qualys VMDR**: A cloud-based vulnerability management, detection, and response service.

### **Network Scanning & Enumeration**

13. **Nmap (Network Mapper)**: The definitive tool for network discovery and security auditing.
14. **Masscan**: The world's fastest Internet port scanner, capable of scanning the entire Internet in minutes.
15. **Zmap**: Another fast single-packet network scanner designed for Internet-wide surveys.
16. **Wireshark**: The world's foremost and widely-used network protocol analyzer.
17. **tcpdump**: A powerful command-line packet analyzer.

### **Web Application Testing**

18. **Burp Suite**: The standard toolkit for web application security testing (Community & Professional editions).
19. **OWASP ZAP (Zed Attack Proxy)**: A free, open-source alternative to Burp Suite, very powerful and active.
20. **SQLmap**: Automates the process of detecting and exploiting SQL injection flaws.
21. **XSStrike**: A advanced XSS detection and exploitation suite.
22. **Commix**: An automated tool that can find and exploit Command Injection vulnerabilities.
23. **Wfuzz**: A web application fuzzer used for brute-forcing parameters and directories.
24. **FFUF (Fuzz Faster U Fool)**: A very fast web fuzzer written in Go.
25. **Dirb / Dirbuster**: Tools used to brute-force directories and file names on web servers.
26. **Gobuster**: A tool used to brute-force URIs, DNS subdomains, and Virtual Host names.
27. **WhatWeb**: A website fingerprinting tool that identifies technologies used on websites.
28. **WPScan**: A black box WordPress vulnerability scanner, similar to JoomScan but for WordPress.
29. **Droopescan**: A plugin-based scanner for CMSs like Drupal, Silverstripe, and WordPress.

### **Exploitation Frameworks & Tools**

30. **Metasploit Framework** (You already have this, it's the king.)
31. **SQLmap** (Also fits here for SQLi exploitation.)
32. **BeEF (The Browser Exploitation Framework)**: Focuses on the client-side, hooking web browsers.
33. **Empire / Starkiller**: A post-exploitation framework that uses PowerShell and Python agents.
34. **Cobalt Strike**: A commercial, post-exploitation tool designed for red teams.

### **Password Attacks**

35. **Hydra**: A parallelized login cracker which supports numerous protocols.
36. **John the Ripper**: A fast password cracker, originally for UNIX but now supports many platforms.
37. **Hashcat**: The world's fastest and most advanced password recovery utility (uses GPU).
38. **Medusa**: Another speedy, parallel, and modular login brute-forcer.
39. **Crunch**: A wordlist generator where you can specify a character set and password length.

### **Post-Exploitation & Privilege Escalation**

40. **Mimikatz**: The famous tool for extracting plaintext passwords, hashes, and Kerberos tickets from memory.
41. **PowerSploit**: A collection of PowerShell modules for post-exploitation.
42. **LinPEAS / WinPEAS**: Incredible scripts that automatically search for possible privilege escalation paths on Linux/Windows.
43. **PsExec**: A light-weight telnet-replacement that lets you execute processes on other systems.
44. **BloodHound**: Uses graph theory to reveal the hidden and often unintended relationships within an Active Directory environment.

### **Wireless Network Testing**

45. **Aircrack-ng**: A complete suite of tools to assess WiFi network security.
46. **Kismet**: A wireless network detector, sniffer, and intrusion detection system.
47. **Wifite**: An automated wireless attack tool.
48. **Reaver**: Implements a brute force attack against WPS to recover WPA/WPA2 passphrases.

### **Social Engineering**

49. **SET (Social-Engineer Toolkit)**: An open-source penetration testing framework designed for social engineering.
50. **Gophish**: An open-source phishing toolkit designed for businesses and penetration testers.
51. **King Phisher**: A tool for testing and promoting user awareness by simulating real-world phishing attacks.

### **Forensics & Incident Response**

52. **Autopsy**: A digital forensics platform and graphical interface to The Sleuth Kit.
53. **Volatility**: An advanced memory forensics framework for extracting digital artifacts from RAM.
54. **FTK Imager**: A forensics tool for creating disk images and previewing evidence.
55. **Binwalk**: A tool for analyzing, reverse engineering, and extracting firmware images.

### **Miscellaneous & Utilities**

56. **Searchsploit**: A command-line search tool for Exploit-DB.
57. **Netcat**: The network Swiss army knife, used for reading from and writing to network connections.
58. **SOCAT**: A more advanced and versatile Netcat variant.
59. **EyeWitness**: A tool designed to take screenshots of websites, RDP services, and open VNC servers.
60. **Unicorn**: A tool for using PowerShell downgrade attacks and injecting Shellcode into memory.
61. **Chisel**: A fast TCP/UDP tunnel, transported over HTTP, secured via SSH. Useful for pivoting.
62. **Impacket**: A collection of Python classes for working with network protocols, containing many useful tools for crafting packets and exploiting Windows services.
